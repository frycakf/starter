=== Environment setup ===
1. Install Node.js
2. Create package.json [npm init]
X. Run [npm install] to install all packages.

=== Packages to install ===
1. live-server
	localy [npm install live-server --save-dev]
	or globaly [npm install live-server -g]
2. node-sass [npm install node-sass --save-dev]
3. concat [npm install concat --save-dev]
4. autoprefixer [npm install autoprefixer --save-dev]
	needs postcss-cli [npm install postcss-cli --save-dev]
5. npm-run-all [npm install npm-run-all --save-dev]


=== Build process ===
1. main.scss
2. style.concat.css
3. style.prefix.css
4. style.comp.css
5. style.css
